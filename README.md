FeedRSS
=======

Basic Reader Feeds

Technical Features
-------------------

The system provide the following features:

* Automatic update of news;
* Search terms in the feed's messages;
* Show graphs of performance;
* Follow your favorite sites;
* See a summary of the news before accessing it;
* Loading on demand;
* Shared your news;
